## 🌟 Unique Features in Our System

### 1. **VHPT Format Store Transactions**
- GRN, STN, SRN following VHPT standards
- Automatic inventory updates
- Company-specific warehouse separation

### 2. **Hajri System**
- Indian construction standard (1, 1.5, 2)
- Manpower reporting with worker types
- DPR integration

### 3. **Drawing Panel Progress Tracking**
- Upload drawings (PDF, DWG, images)
- Mark panels on drawings
- Track progress percentage
- Visual progress overlay (planned)

### 4. **Equipment Breakdown Deductions**
- Automatic calculation based on breakdown hours
- Rate per day/sqm support
- Net amount after deductions

### 5. **3-Level Expense Approval**
- Store Manager → Operation Manager → Head/Accounts
- Bill upload (Kaccha/Pakka)
- Selfie capture
- Approval workflow tracking

### 6. **Multi-Company Warehouse System**
- Separate warehouses for VHPT and VHSHREE
- Common warehouse support
- Access control per company
